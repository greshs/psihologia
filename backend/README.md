Backend API для управления контентом курса МАК

Простой Node.js бэкенд для управления контентом сайта.

## Установка

```bash
cd backend
npm install
```

## Запуск

```bash
npm start
```

Сервер запустится на порту 3001 (или PORT из переменных окружения).

## Использование

### API Endpoints

- `GET /api/content` - Получить весь контент
- `PUT /api/content` - Обновить весь контент
- `PATCH /api/content` - Обновить часть контента

### Админка

Откройте в браузере: `http://localhost:3001/admin`

Здесь вы можете редактировать весь контент сайта через удобный интерфейс.

## Данные

Данные хранятся в файле `data/content.json` в формате JSON.

## Деплой

Для деплоя на бесплатный хостинг (например, Railway, Render, или Vercel):

1. Создайте аккаунт на хостинге
2. Подключите репозиторий
3. Укажите команду запуска: `npm start`
4. Укажите порт из переменной окружения `PORT`

После деплоя обновите `REACT_APP_API_URL` в React приложении на URL вашего бэкенда.
